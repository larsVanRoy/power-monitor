The monitor that keeps track of generated power.

This monitor allows you to properly track the generated power. All data is stored within a database up to a full year. The data can be displayed over the ranges of a week, a month and a year. Specific data can be requested (if within the range of the past 365 days). Along with the visual display of the data there are statistics ranging over the entire duration in which data was tracked.



Press the button below to update the database, this operation will only work in case the required settings, being the ip and start-date variables, have been set properly. This process might take a few seconds, as it will also refresh the database and generate all plots.
